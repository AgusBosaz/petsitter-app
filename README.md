# PETSITTER-APP

- # DESCRIPCION DEL PROYECTO
  Petsitter, es una app para el cuidado de animales.

  El problema que busca resolver, es resolver lo mas rapido posible con quien dejamos a nuestras mascotas.

  Los usuarios principales ademas del Admin, van a ser los CUIDADORES y los DUEÑOS DE MASCOTAS.

- # FUNCIONALIDADES ACTUALES
  Actualmente tenemos 3 funcionalidades, buscar-cuidadores, login-cuidadores y registro-cuidadores. Ademas de del index.

- # TECNOLOGIAS UTILIZADAS
  Las tecnologias usadas hasta el momento son: HTML, CSS y JavaScript.
